Welcome to the fundamentals of Networking, Cyber Security and Information!!

This website is going to include the basic principles of information technologies and how they can be used to connect users all over the world in a safe and secure fashion that allows users to exchange information with each other without exposing them to any dangers.

Knowing how to protect yourself when roaming the web is an aboolutely crucial skill to have in todays society, given the fact that almost everything is online now, whether it be banking, medical records or other sensititve information.

As a result of this, Cyber Security activity has increased exponentially over the past ten years and will only continue to do so as newer technology is developed. Given the fact you are currently using this website it is safe to assume you want to learn more about how to stay safe when online.

Most of the Information you see on this website was gathered from various sources by a small team of undergraduate students currently studying cyber security at the University of Brighton.
